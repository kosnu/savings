# Category Budget Domain Rules

## Rules

- カテゴリ別予算はBookとカテゴリに属する。
- カテゴリ別予算は対象月の月初日を有効開始日として保存する。
- カテゴリ別予算の対象年と対象月は有効開始日から決まる。
- 同一Book・同一カテゴリ・同一年・同月のカテゴリ別予算は重複できない。
- カテゴリ別予算レコードは金額あり予算または予算なし状態を表す。
- 金額あり予算は `amount` を持ち、予算なし状態は `amount` を持たない。
- 0円予算は金額あり予算であり、予算なし状態とは別の状態として扱う。
- 対象月の有効状態は、対象月末日以前に開始した同一Book・同一カテゴリのレコードのうち最新の1件で決まる。
- 最新の有効状態が予算なしの場合、さらに過去の金額あり予算は対象月に復活しない。
- 当月に有効な過去月開始の金額あり予算を更新または削除する場合、過去月の表示を変えずに当月開始の新しい状態として保存する。
- カテゴリ設定で予算を削除する操作は物理削除ではなく、当月以降を予算なしにする状態変更として扱う。
- カテゴリ作成時の予算未入力は未設定として扱い、カテゴリ別予算レコードを作らない。
- カテゴリ別予算の金額は金額ドメインのルールに従う。
- カテゴリ別支出合計は、対象月の支払い合計として扱う。
- カテゴリ別サマリでは金額あり予算が有効な場合だけ予算額を表示する。
- 予算なし状態、未設定状態、未分類サマリでは予算額を表示しない。
